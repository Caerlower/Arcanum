# Arcanum — Private Swaps

Privacy-first DeFi protocol for the Aleo Privacy Buildathon. Private token swaps with hidden balances, order size, and execution intent.

## Tech stack

- Vite, TypeScript, React
- shadcn-ui, Tailwind CSS

## Develop locally

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview production build
- `npm run lint` — run ESLint
- `npm run test` — run tests
